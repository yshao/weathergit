DEPENDENCIES
- python 2.5 or 2.6 or 2.7
  Not yet tested on python 2.7 but should work :)
  http://python.org

- simplejson 2.1.2 at least Revision r232
  http://code.google.com/p/simplejson/source/checkout

- ordereddict 0.4.4
  http://www.xs4all.nl/~anthon/ordereddict

- chardet 2.0.1
  Tested only with chardet version 2.0.1, may work with versions prior to 2.0.1
  http://chardet.feedparser.org

- PyQt4 4.4.4 or newer
  Tested with PyQt4 4.4.4 and 4.7.3, may work with versions prior to 4.4.4
  http://www.riverbankcomputing.co.uk

- QScintilla2
  Tested with QScintilla2 2.3.2 and 2.4.4, may work with versions prior to 2.3.2)
  http://www.riverbankcomputing.co.uk/software/qscintilla/download
